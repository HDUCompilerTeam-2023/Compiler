# Change log

## 0.9.0

完成所有文法规则规约时的输出

## 0.8.0

不使用 bison 的优先级消除 if-else 规则的二义性，新方案将 while 后的语句作了分类
基于语句的 if-else 是否配对，while 后跟配对的语句可推出配对的语句，后跟不能配对的语句则推出不配对的语句

其他特性

- 在不可使用逗号表达式的规则中使用赋值表达式
- 目前暂定不可使用逗号表达式的部分
  - 下标定义
  - 变量初始化的值
- 函数形参的下标的第一项的值可选为空或给出，例如，`int f(int a[][2])`与`int f(int a[2][2])`都是正确的
